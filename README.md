# MFCApplication1
项目说明文档
=================
这是一个使用Microsoft Foundation Class (MFC) 库开发的Windows应用程序示例项目。该项目展示了如何创建一个基本的MFC应用程序，包括窗口创建、消息处理和用户界面设计。
注意事项：
1. 环境要求
   - 操作系统：Windows 10或更高版本
   - 开发工具：Microsoft Visual Studio 2019或更高版本
   - MFC库：确保在Visual Studio安装时选择了MFC组件
2.本项目Gmae文件夹内未使用预编译头
	- 请手动关掉预编译头的设置
	- 右键Game文件夹内的源文件(.cpp)
	- 属性 - 配置属性 - C/C++ - 预编译头 - 不使用预编译头(/Y-)
3. 编译和运行
   - 打开Visual Studio，加载项目文件（.sln）。
   - 配置项目属性，根据需要设置调试或发布模式。
   - 编译项目，确保没有错误。
   - 运行应用程序，测试

   功能和界面
1. 按d是调试功能，能看见一些调试信息
2. 行动作由箭头键控制，空格也可以跳跃
