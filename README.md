# MFC 超级马里奥风格平台游戏

这是一个使用 C++ 和 Microsoft Foundation Classes (MFC) 框架开发的经典 2D 平台游戏，灵感来源于《超级马里奥兄弟》。

## ✨ 功能特性

- **玩家控制**: 实现了马里奥的左右移动、跳跃和基本的物理效果（重力、加速度）。
- **角色状态**: 马里奥拥有多种状态，例如从小变大。
- **可切换皮肤**: 玩家可以在默认的马里奥皮肤和初音未来（Miku）皮肤之间切换。
- **动画系统**: 使用精灵图（Sprite Sheet）实现行走和站立的动画。
- **碰撞检测**: 实现了与砖块、水管等游戏元素的精确碰撞检测。
- **关卡设计**: 包含一个简单的游戏关卡，展示了地面、砖块和水管的布局。
- **镜头跟随**: 游戏镜头会平滑地跟随玩家角色移动。

## 🛠️ 技术栈

- **语言**: C++
- **框架**: Microsoft Foundation Classes (MFC)
- **开发环境**: Visual Studio

## 🚀 如何开始

### 环境要求

- **Visual Studio**: 需要安装 Visual Studio（推荐 2019 或更高版本）。
- **工作负载**: 确保已安装 **"使用 C++ 的桌面开发"** 工作负载，并勾选了 **"C++ MFC"** 组件。

### 构建和运行

1.  **克隆仓库**:
    ```bash
    git clone https://github.com/HC68-MAX/MFCApplication1.git
    ```
2.  **打开解决方案**:
    在 Visual Studio 中打开项目根目录下的 `.sln` 解决方案文件。

3.  **设置启动项**:
    在解决方案资源管理器中，右键点击 `MFCApplication1` 项目，选择 “设为启动项目”。

4.  **编译和运行**:
    按下 `F5` 或点击工具栏上的 "本地 Windows 调试器" 按钮来编译和运行游戏。

## 🎮 控制方式

- **向左移动**: `A` 键或 `←` (左箭头)
- **向右移动**: `D` 键或 `→` (右箭头)
- **跳跃**: `W` 键或 `↑` (上箭头) 或 `空格键`
- **切换皮肤**: `Q` 键 (在马里奥和初音未来之间切换)
- **切换状态**: `E` 键 (在小马里奥和大马里奥之间切换)
